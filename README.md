# NZBOT — Protect Your App Against Non-Humans

![NZBOT Preview](img/Screenshot%202026-04-08%20202525.png)

> A sleek, cyberpunk-styled landing page for a bot detection & protection service.

---

## What is NZBOT?

NZBOT is a bot-free mobile app protection platform. The landing page showcases the product with a dark futuristic UI, animated particles, cyber grid lines, and an interactive 3D Spline scene.

## Tech Stack

- HTML5 / CSS3 / Vanilla JS
- [Spline](https://spline.design/) — interactive 3D model viewer
- [Orbitron](https://fonts.google.com/specimen/Orbitron) — Google Font for the logo
- Canvas API — custom floating particle system

## Features

- Animated gradient logo with `Orbitron` font
- Floating particle canvas background (indigo / violet / sky palette)
- Cyber grid overlay with horizontal & vertical lines
- Radial glow accents
- Responsive hero layout with CTA buttons
- Embedded Spline 3D scene

## Getting Started

Just open `indext.html` in your browser — no build step needed.

```bash
# Clone the repo
git clone <your-repo-url>

# Open in browser
open indext.html
```

## Project Structure

```
├── indext.html       # Main HTML entry point
├── style.css         # All styles (dark theme, layout, animations)
├── script.js         # Canvas particle system
└── img/              # Screenshots & assets
```
